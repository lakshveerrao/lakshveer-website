# Drishtikon
*Project*

## Summary
Drishtikon Yantra is a Raspberry Pi-based wearable assistive device designed for visually impaired users. The device uses computer vision and text-to-speech technology to describe the surrounding environment in real-time. Lakshveer developed and publicly demonstrated this social impact project as part of his hardware and accessibility work.

## Status & Timeline
- **2025-08**: Public demo at South Park Commons pitch event in Bangalore alongside CircuitHeroes [[sig-spc-demo-2025]]
- **2026-01**: Project documented as active wearable device [[sig-drishtikon-2026]]

## Technical Details
- **Hardware**: Raspberry Pi-based wearable form factor
- **Core Technology**: Computer vision processing paired with text-to-speech output
- **Purpose**: Real-time environmental description for users with visual impairments
- **Category**: Assistive technology, accessibility hardware

## Impact
Drishtikon represents Lakshveer's work in social impact technology and demonstrates his ability to build functional hardware prototypes for underserved user populations. The project earned presentation opportunities at South Park Commons, a notable tech community space.
