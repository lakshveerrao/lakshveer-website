# computer-vision
*Domain*

## Summary
Lakshveer has built multiple computer vision systems spanning assistive technology, gesture recognition, and agent-based control. Projects include Drishtikon Yantra (wearable vision device for visually impaired), MotionX (full-body motion tracking for gaming), and AI agents that interpret visual input for hardware control. Work began publicly by 2025-08 and accelerated through 2026.

## Evidence
**Drishtikon Yantra** — Raspberry Pi-based wearable assistive vision device for visually impaired users. Publicly demoed at South Park Commons (2025-08) [[sig-spc-demo-2025]] and detailed in YouTube video (2026-01) [[sig-drishtikon-2026]]. Earned Special Mention at Param × Vedanta Makeathon. System processes visual input in real-time on edge hardware.

**MotionX** — Full-body motion-control gaming system built at RunTogether Hackathon by Runable (2026-01) [[sig-motionx-launch-2026]]. Uses computer vision for gesture recognition without external sensors. Won Special Mention. Demonstrates integration of CV with interactive systems.

**AI Agent Hardware Control** — YouTube Short (2026-03) [[sig-yt-short-telegram-2026]] shows AI agent controlling ESP32 hardware via Telegram messaging. Agent interprets visual commands or states through computer vision pipeline, translating to hardware instructions.

## Trajectory
- **2025-08**: First public computer vision demo (Drishtikon Yantra) at South Park Commons
- **2026-01**: Two major CV projects launched — Drishtikon Yantra detailed release + MotionX gaming system
- **2026-03**: Integration of CV with AI agents for hardware control via messaging interfaces

Pattern: Consistent focus on practical, edge-deployable computer vision (Raspberry Pi, ESP32). Applications span accessibility (Drishtikon), gaming (MotionX), and agent-based control systems. All projects emphasize real-time processing and physical-world interaction rather than cloud-dependent solutions.
