*Domain*

## Summary
Lakshveer applies computer vision to assistive technology and interactive systems. Primary work includes Drishtikon Yantra (wearable vision device for visually impaired) and MotionX (gesture-controlled gaming). Uses Raspberry Pi and AI models for real-time object detection and motion tracking.

## Evidence
**Drishtikon Yantra** — Raspberry Pi-based wearable assistive vision device for visually impaired users. Earned Special Mention at Param × Vedanta Makeathon (2026-01). Demonstrated publicly at South Park Commons (2025-08). [[sig-drishtikon-2026]] [[sig-spc-demo-2025]]

**MotionX** — Full-body motion-control gaming system using computer vision for gesture recognition. Built at RunTogether Hackathon by Runable. Special Mention Winner (2026-01). [[sig-motionx-launch-2026]]

**AI Agent Hardware Control** — YouTube Short showing CV-based AI agent controlling ESP32 hardware via Telegram messaging interface (2026-03). [[sig-yt-short-telegram-2026]]
